# Python_Starter_CyberBionic_Systematics
Цей репозиторій містить матеріали та практичні роботи за програмою курсу **«Python Базовий»**. 
Курс спрямований на поглиблення знань мови Python, вивчення об'єктно-орієнтованого програмування та роботу зі складними структурами даних

## Програма курсу

### Урок 1 --- ООП. Успадкування

У цьому уроці студенти знайомляться з основами об'єктно-орієнтованого
програмування в Python. Розглядається, що таке класи та об'єкти, як вони
створюються і для чого використовуються у реальних проєктах. Особлива
увага приділяється принципу успадкування: як створювати дочірні класи,
як перевизначати методи та як будувати ієрархію класів.

------------------------------------------------------------------------

### Урок 2 --- Інкапсуляція

На цьому уроці розглядається один із ключових принципів ООП ---
інкапсуляція. Студенти дізнаються, як обмежувати доступ до змінних та
методів у класах, що таке public, protected і private змінні в Python.
