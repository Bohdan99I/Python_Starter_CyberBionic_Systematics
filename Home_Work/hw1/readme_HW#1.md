# Домашнє завдання

## Завдання 1

Встановіть **Python** та **PyCharm** (або іншу IDE з підтримкою Python, яка вам зручна).  
Дослідіть налаштування та налаштуйте середовище розробки під себе:

- Оберіть зручну для очей кольорову схему редактора
- Налаштуйте шрифт
- Увімкніть або вимкніть:
  - відображення номерів рядків
  - підсвічування поточного рядка
  - відображення роздільників між ділянками коду
- За потреби змініть інші параметри для комфортної роботи

---

## Завдання 2

Створіть **скрипт мовою Python**, використовуючи звичайний текстовий редактор  
(можна також використовувати редактор коду, такий як Sublime Text або Notepad++).

1. Збережіть файл з розширенням `.py`
2. Запустіть його **через консоль**
3. Запустіть його **подвійним клацанням** у Провіднику Windows
4. Знайдіть спосіб зробити так, щоб вікно консолі **не закривалося одразу** після запуску скрипта подвійним кліком

> 💡 Підказка: можна використати `input()` наприкінці скрипта або інші способи, які ви вже знаєте.

---

## Завдання 3

Відкрийте **IDLE** (встановлюється разом з Python).

Поекспериментуйте зі звичайними арифметичними виразами. Спробуйте задати ім'я значенню будь-якого виразу. Спробуйте вивести значення виразу з текстом, що пояснює, за допомогою функції print, використовуючи як імена, так і безпосередньо вирази в якості параметрів функції.
