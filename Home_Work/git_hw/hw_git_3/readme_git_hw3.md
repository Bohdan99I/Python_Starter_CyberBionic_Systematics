# README.md

## Перелік завдань

### Завдання 1
Проведіть інтеграцію IDE та GIT.

### Завдання 2
За допомогою вашої IDE створіть проєкт з 5 файлів та надішліть його на віддалений репозиторій.

### Завдання 3
Створіть декілька репозиторіїв з файлами ваших минулих проєктів двох видів:
- public,
- private.

Для приватного репозиторію:
- надати доступ 5 колегам та помічнику тренера,
- отримати від них код-рев'ю.

### Завдання 4
Кожен з 5 колег та помічник тренера повинні:
- створити нову гілку,
- додати декілька файлів,
- зробити коміт,
- надіслати зміни на віддалений репозиторій.

### Завдання 5
Виконати злиття доданих гілок та видалити зайві.

### Завдання 6 (командна робота)
Створіть у команді (2–3 учні) програму-калькулятор.

1. Перший учень створює проект/файл/клас та завантажує на віддалений сервер.
2. Кожний наступний учень:
   - клонує проєкт,
   - додає свою частину програми,
   - робить коміти,
   - надсилає зміни назад на сервер.
